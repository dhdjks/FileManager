# 文件管理器 - 规格文档

## 1. 项目概述

- **项目名称**: FileManager
- **类型**: Android 原生应用 (Kotlin)
- **核心功能**: 现代化界面的文件管理器，支持文件浏览、复制、粘贴、剪切、删除、重命名等基础操作

## 2. 技术栈

- **语言**: Kotlin
- **最低 SDK**: 26 (Android 8.0)
- **目标 SDK**: 34 (Android 14)
- **UI 框架**: Jetpack Compose + Material Design 3
- **架构**: MVVM + Clean Architecture
- **异步**: Kotlin Coroutines + Flow
- **依赖注入**: Hilt

## 3. 功能列表

### 3.1 文件浏览
- 显示内部存储文件列表
- 支持进入子文件夹
- 支持返回上级目录
- 显示文件图标、名称、大小、修改时间

### 3.2 文件操作
- **复制 (Copy)**: 复制文件/文件夹到剪贴板
- **粘贴 (Paste)**: 将剪贴板内容粘贴到当前位置
- **剪切 (Cut)**: 移动文件/文件夹到剪贴板
- **删除 (Delete)**: 删除文件/文件夹
- **重命名 (Rename)**: 修改文件/文件夹名称

### 3.3 排序与视图
- 按名称排序
- 按大小排序
- 按修改时间排序
- 列表视图/网格视图切换

## 4. UI/UX 设计方向

### 4.1 视觉风格
- Material Design 3 (Material You)
- 动态颜色主题
- 圆角卡片设计
- 现代化阴影和动画

### 4.2 配色方案
- 主色调: 蓝色系 (#1976D2)
- 背景: 浅色主题 / 深色主题 自动适应系统
- 文件夹图标: 黄色 (#FFC107)
- 文件图标: 灰色系，根据文件类型变化

### 4.3 布局结构
- **顶部**: 标题栏 + 路径导航
- **中部**: 文件/文件夹列表 (可滚动)
- **底部**: 操作栏 (复制/粘贴/剪切/删除 快捷操作)
- **悬浮**: FAB 按钮 (新建文件夹)

### 4.4 交互设计
- 长按进入多选模式
- 单击打开文件/文件夹
- 右滑快捷删除
- 底部弹窗显示操作菜单
